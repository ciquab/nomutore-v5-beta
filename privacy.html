NOMUTORE

Privacy Policy

施行日: 2026年2月18日

NOMUTORE（以下「本アプリ」）は、ユーザーのプライバシーを最大限に尊重して設計されたウェブアプリケーションです。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

本アプリはサーバーにユーザーデータを保存しません。すべてのデータはお使いのデバイス上にのみ保存されます。

1. 収集する情報

1-1. デバイス上に保存されるデータ

本アプリは以下のデータをお使いのブラウザの IndexedDB および localStorage に保存します。これらのデータは外部サーバーには送信されません。

データの種類 内容 目的
飲酒記録 ビールの銘柄名、スタイル、カロリー、メモ カロリー収支の計算
運動記録 運動種目、時間、消費カロリー カロリー収支の計算
デイリーチェック 健康チェック項目の回答 健康状態の記録
プロフィール 体重、身長、年齢、性別 基礎代謝の計算
設定 テーマ、通知設定、期間モード等 アプリのカスタマイズ

1-2. Google ドライブに保存されるデータ

ユーザーが明示的にバックアップ操作を行った場合に限り、上記のデータが Google ドライブに保存されます。バックアップには以下の情報が含まれます。

  • 飲酒・運動記録、デイリーチェック、期間アーカイブ
  • アプリの各種設定
  • デバイス情報（User-Agent 文字列）

バックアップファイルはユーザーの Google ドライブ上に nomutore_backup.json という名前で保存され、本アプリからのみアクセス可能です。

1-3. プッシュ通知

ユーザーがプッシュ通知を有効にした場合、通知配信に必要なサブスクリプション情報（エンドポイント URL）がサーバーに保存されます。この情報は通知の配信のみに使用されます。

2. Google API の利用について

本アプリは Google Drive API を使用してバックアップ機能を提供しています。

2-1. 使用する OAuth スコープ

  • https://www.googleapis.com/auth/drive.file — 本アプリが作成したファイルに対してのみアクセスする、制限付きスコープです。ユーザーの Google ドライブ内の他のファイルには一切アクセスしません。

2-2. Google API Services User Data Policy への準拠

本アプリによる Google API を通じて取得した情報の使用および他のアプリへの転送は、Google API Services User Data Policy（Limited Use requirements を含む）に準拠しています。

具体的に、本アプリは以下を遵守します。

  • Google ドライブから取得したデータは、バックアップの復元のみに使用します
  • 取得したデータを第三者に販売、貸与、または提供しません
  • 広告配信やターゲティングの目的で使用しません
  • ユーザーが明示的に同意または指示した場合を除き、データを転送しません
  • 人間によるデータの閲覧は、セキュリティ上の目的、法令遵守、またはユーザー本人の同意がある場合に限定されます

3. データの共有

本アプリは、以下の場合を除き、ユーザーの情報を第三者と共有しません。

  • ユーザーが明示的にバックアップ操作を行い、Google ドライブに保存する場合
  • ユーザーが明示的にデータのエクスポート操作を行う場合
  • 法令に基づく要請がある場合

4. データの保管とセキュリティ

  • ローカルデータはブラウザの標準的なストレージ機能（IndexedDB、localStorage）により保護されます
  • Google ドライブとの通信は HTTPS により暗号化されます
  • OAuth 2.0 トークンはブラウザのメモリ上にのみ保持され、永続的に保存されません

5. データの削除

ユーザーは以下の方法でデータを削除できます。

  • アプリ内: 設定タブの「全データを削除」ボタンで、すべてのローカルデータを削除
  • ブラウザ: ブラウザの設定からサイトデータを消去
  • Google ドライブ: Google ドライブから nomutore_backup.json を直接削除

6. Cookie・トラッキング

本アプリは Cookie を使用しません。アクセス解析ツールや広告トラッキングも一切使用していません。

7. 第三者サービス

本アプリは以下の第三者サービスを利用しています。各サービスのプライバシーポリシーもご確認ください。

  • Google Drive API: バックアップ機能 — Google プライバシーポリシー
  • Google Cloud Functions: プッシュ通知の配信
  • CDN サービス（jsDelivr, unpkg）: ライブラリの配信（ユーザーデータは送信されません）

8. 子どものプライバシー

本アプリは飲酒に関連するコンテンツを含むため、20歳未満の方を対象としていません。20歳未満の方が本アプリを使用していることが判明した場合、該当するデータは速やかに削除されます。

9. ポリシーの変更

本ポリシーは必要に応じて改定されることがあります。重要な変更がある場合は、アプリ内で通知します。
